# Project 1

# Secure Login and User Profile Web Application

• Developed an OIDC-integrated Single Page Application using JavaScript and Okta to handle secure user
authentication and session management.

• Configured custom Identity Claims in Okta to dynamically extract and display user metadata, including roles,
names, and email addresses.

• Managed Okta App Integration by setting up Client IDs and Redirect URIs to ensure a seamless handshake
between the website and the identity provider.

• Implemented a dynamic UI that captures and renders personalized profile data from the authentication response
upon successful login.

# Project 2

# Role-Based Access Control (RBAC) System Implementation

• Engineered a Role-Based Access Control (RBAC) System using Okta to manage "Admin" and "User" groups with
distinct permission levels.

• Defined Custom Scopes and Policies to restrict sensitive actions, such as delete:data, ensuring only authorized
Admin members could access specific API privileges.

• Executed Authorization Code Flows using OIDC Debugger and Postman to exchange codes for Access Tokens
and validate security claims.

• Performed Token Lifecycle Audits by decoding JWTs to verify that group-based policies correctly mapped scopes
and permissions to the appropriate users.

# Project 3

# SAML 2.0 Single Sign-On (SSO) Implementation

• I implemented Single Sign-On using SAML 2.0 by integrating Okta as the Identity Provider with a Service
Provider test application.

• Configured Entity ID, Single sign-on URL, and exchanged metadata to establish secure authentication

• Assigned users and validated seamless login without separate application credentials

• Analyzed SAML requests and responses using SAML Tracer to troubleshoot authentication flow
